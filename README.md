## 📖 About The Project

This **Employee Management System** is a complete CRUD (Create, Read, Update, Delete) application that allows you to manage employee records efficiently. The backend is developed using **Spring Tool Suite 4 (STS)** with Spring Boot, and the frontend is built with vanilla HTML, CSS, and JavaScript.

### Why Spring Tool Suite (STS)?
- STS is specifically designed for Spring Boot development
- Built-in Spring Boot dashboard for easy management
- Automatic dependency management
- Embedded server support (Tomcat)
- Easy debugging and testing

### Key Functionalities:
- ✅ Add new employees with detailed information
- ✅ View all employees in a structured table
- ✅ Search employees by name
- ✅ Filter employees by department
- ✅ Edit existing employee details
- ✅ Delete employees with confirmation
- ✅ Real-time updates without page refresh

## ✨ Features

### Backend Features (Spring Boot)
- RESTful API architecture
- JPA/Hibernate for database operations
- MySQL database integration
- Validation and error handling
- Service layer for business logic
- Repository pattern for data access
- CORS configuration for frontend access

### Frontend Features
- Responsive design (works on all devices)
- Interactive modal for editing
- Real-time search functionality
- Department-wise filtering
- Loading states and spinners
- User-friendly error messages
- Confirmation dialogs for delete operations
- Clean and modern UI with Font Awesome icons
